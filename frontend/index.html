InovaHub

Plataforma de inovação com cadastro, login por ID, ideias, missões, chat e ranking.

Cadastro

Crie um usuário informando nome e e-mail. Após o cadastro, o sistema mostrará o ID do usuário. Guarde esse ID para realizar login.

Login

Entre usando o ID do usuário criado no cadastro ou retornado no Swagger pelo endpoint POST /api/users.

Dica: no Swagger, ao criar um usuário, copie o campo id da resposta e use aqui no login.

Swagger: /api-docs
